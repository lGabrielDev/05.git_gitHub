## Introduction <img src="https://cdn-icons-png.flaticon.com/512/1436/1436664.png" alt="imagem" width="50px" align="center">
O `git branch` command serve para listas todas as branch ou criar uma branch.

<hr>
<br>


<!--
git checkout

Troca para outra branch;


1- Crie uma branch:
        git branch criar_botao;

2- Liste as branches:
        git branch;

3- Troque para a branch criada: 
        git checkout criar_botao;

Easy!


Lembrando!!

Digamos que eu esteja na branch main, faço alteracao em um arquivo e dou um commit, e troco de branch.

Se eu der um push, o commit será enviado para a branch atual;

Entao... Quando eu troco de branch, todos os commits vêm junto;


As vezes, para acessar branches de outros devs, devemos falar o caminho completo da branch:
	git checkout origin/branchTal;

-->
