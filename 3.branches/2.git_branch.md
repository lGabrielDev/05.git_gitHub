<!--
git branch -> Lista as branches;

git branch branchTal -> Cria uma branch;

git branch -d branchTal -> Deleta uma branch;


Vamos praticar no repositorio "repositorio_1";



1- Liste as branches:
        git branch;
        
        Por enquanto, só tem a branch main/master;


2- Crie uma branch:
        git branch criar_novo_botao;

3- Liste novamente:
        git branch;

4- Delete a branch criada:
       git branch -d criar_novo_botao;

       
Geralmente não é deletado as branches. Apenas se seu chefe pedir;


Lembrando!!!

Eu sempre crio uma branch a partir da branch main/master;

Entao, antes de criar uma branch, voce precisa verificar se está na branch main/master; Aí sim, podemos criar nossa branch;
-->
